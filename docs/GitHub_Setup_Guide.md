## 📊 Project Board Usage

### Column Flow

```text
Backlog → To Do → In Progress → In Review → Done 
  ↓         ↓          ↓            ↓         ↓ 
Not start  Ready     Working     Need check Complete
```

### How to Use

1. **Backlog** - Nơi tất cả issues được tạo
2. **To Do** - Khi issue được assigned & ready
3. **In Progress** - Khi bắt đầu làm việc
4. **In Review** - Khi hoàn thành, chờ review
5. **Done** - Khi approved & closed

## 🏷️ Label Usage Guide

### Lab Labels
- Use **Lab1-5** to group issues by phase
- Each issue should have exactly **1 Lab label**

### Category Labels
- Use to indicate type of work
- **planning** - For planning/analysis tasks
- **documentation** - For writing/doc tasks
- **execution** - For doing/implementation
- **controlling** - For monitoring/review
- **closing** - For closure/lessons learned

### Priority Labels
- Use **1 priority label** per issue
- **priority-critical** - Must do, blocks other work
- **priority-high** - Important, do soon
- **priority-medium** - Normal priority
- **priority-low** - Nice to have

## 💡 Tips & Best Practices

### 1. Naming Conventions
✅ DO:
```text
[Lab X] Task Name 
[Lab 1] Identify Project Stakeholders 
[Lab 2] Create Work Breakdown Structure
```

❌ DON'T:
```text
Fix bug 
Task 1 
Something important
```

### 2. Assignment
- Assign issues to specific people
- One person = One issue (or pair)
- Update assignee when transferring work

### 3. Comments & Updates
- Comment on issue for progress
- Link to documents being worked on
- Mention @person for notifications

### 4. Closing Issues
- Mark PR/commit as closing issue
- Use "Closes #123" in commit message
- Update issue description when done

### 5. Board Management
- Move card to **In Progress** when starting
- Move to **In Review** when done
- Move to **Done** when approved

## 📞 Troubleshooting

### Issue not appearing on board?
- Check if issue is in correct repo
- Verify issue is not in draft state
- Refresh browser page
- Try adding issue manually again

### Label not showing?
- Verify label name (case-sensitive)
- Check label color visibility
- Try refreshing page
- Re-create label if needed

### Milestone date not changing?
- Click milestone to edit
- Update due date field
- Save changes
- Verify date updated

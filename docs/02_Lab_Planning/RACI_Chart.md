# RACI Chart - Lab 2

## 📊 RACI Matrix Definition
- **R** = Responsible (Does the work)
- **A** = Accountable (Final authority, approval)
- **C** = Consulted (Input/feedback required)
- **I** = Informed (Updated on progress)
